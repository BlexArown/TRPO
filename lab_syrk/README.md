# ТРПО lab with syrk

## Описание
Лабораторная работа по ТРПО.
Реализуется собственная версия операции SYRK из BLAS Level 3
с поддержкой типов float и double.

## Что должно быть сделано
- собственная реализация SYRK;
- тесты корректности;
- тесты производительности;
- сравнение с OpenBLAS;
- запуск на 1, 2, 4, 8 и 16 потоках;
- вычисление относительной производительности;
- вычисление среднего геометрического по 10 запускам.

## Структура проекта
- `include/` — заголовочные файлы
- `src/` — реализация
- `tests/` — тесты корректности
- `bench/` — тесты производительности
- `build/` — папка сборки
